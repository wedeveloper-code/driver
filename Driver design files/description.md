# Описание контент-файлов — tricountydrivingacademy.com

## Источник данных

Все данные извлечены из архива `tricountydrivingacademy.com.zip`, который содержит полную копию сайта Tri-County Driving Academy, Inc. — учебного центра профессиональной подготовки водителей, операторов тяжёлой техники и дизельных механиков, расположенного в Ливане, штат Вирджиния (1714 US Hwy 19, Hansonville, VA).

Сайт был создан на платформе Go Daddy Website Builder (Starfield Technologies) и содержит как обновлённые страницы (основные `.html` файлы с современным дизайном), так и устаревшие страницы (`.htm` файлы и старые `.html` файлы с предыдущей версией сайта).

---

## Структура файлов

### 1. main-pages-content.md — Контент основных страниц

**Источник:** 8 основных страниц сайта (текущая активная версия)

| Файл | Описание |
|------|----------|
| `index.html` | Главная страница — обзор программ обучения, анонсы |
| `about-us.html` | О школе — история, персонал, инструкторы, маршрут |
| `contact-us.html` | Контакты — форма обратной связи, адрес, карта, галерея фото |
| `commercial-driver-training-class--a--cdl.html` | Программа CDL — класс "A" коммерческие водительские права |
| `certified-heavy-equipment-training.html` | Обучение на тяжёлой технике — бульдозеры, экскаваторы |
| `diesel-mechanics-heavy-truck-maintenance.html` | Дизельная механика — 11-месячная программа |
| `nccer-training.html` | Обучение NCCER — национальная сертификация |
| `driver-improvement.html` | Курсы повышения квалификации водителей |

**Что извлечено с каждой страницы:**
- Тег `<title>` (заголовок вкладки браузера)
- Мета-тег `description` (описание для поисковых систем)
- Заголовки H1, H2, H3, H4
- Список изображений с указанием типа (локальные / внешние) и alt-текста
- Полный текстовый контент страницы

**Как использовать:** Контент из этого файла будет непосредственно интегрирован в соответствующие страницы нового дизайна. Каждый блок содержит всё необходимое для наполнения новой страницы: заголовки, описания, текст программ и ссылки на изображения.

## Примечания по использованию

1. **Текстовый контент** в файлах содержит весь видимый текст со страниц, включая навигационные элементы и повторяющиеся блоки. При интеграции в новый дизайн следует использовать основной контент и игнорировать навигационные повторы.

2. **Метаданные** (title, description) следует обновить при создании нового дизайна, но исходные значения сохранены как ориентир для SEO.

3. **Цены на программы** могут быть устаревшими — рекомендуется подтвердить актуальные цены перед публикацией.

4. **Программа Fiber Optics** присутствует только на старых страницах — необходимо уточнить, актуальна ли она для нового сайта.
